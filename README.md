# Ragdalion Next-Gen Corporate Website

Website profil perusahaan resmi PT Ragdalion Revolusi Industri yang modern, profesional, dan berkinerja tinggi, dibuat dengan React (Vite), Tailwind CSS, dan Lucide Icons.

## 🚀 Keunggulan UI/UX (Linear & Vercel Inspired)
- **Deep Space Theme:** Kombinasi latar gelap solid (`#08090a`) dengan Glassmorphism cards (`rgba(255,255,255,0.02)`) dan border putih tipis untuk visual yang elegan dan berkelas.
- **Tight Typography:** Menggunakan font Inter dengan rendering OpenType feature settings `"cv01", "ss03"` untuk tekstur teks yang sangat geometris.
- **Interactive Solutions:** Tabs produk terintegrasi dengan landing page Lovable masing-masing untuk konversi prospek yang maksimal.
- **Consultation Tracker & ISO 27001 Log:** Form pemesanan konsultasi bisnis yang aman dan terintegrasi dengan basis data Sales CRM lokal (LocalStorage) untuk kepatuhan tata kelola data.

## 🛠️ Tech Stack & Dependencies
- **Core:** React 18, Vite 6, Tailwind CSS 4
- **Navigation:** React Router DOM 6
- **Icons:** Lucide React

## 📂 Struktur Halaman
1. **Home:** Hero section berpendar, Partners Grid, Industries, 9 Core Products, dan 4 Core Services.
2. **About Us:** Profil detail tim (88 karyawan, ISO 27001 & ISO 9001), Visi Misi, dan pilar kolaborasi bisnis.
3. **Products & Services:** Showcase detail produk interaktif dengan direct redirection link.
4. **Success Stories:** Use-case filterable (AI, Production, Maintenance, Hardware) dengan visual *Problem -> Solution -> Impact* yang informatif.
5. **News & Update:** Blog edukasi digitalisasi pabrik & Factory DX.
6. **Contact Us:** Booking form konsultasi interaktif dengan FAQ dan Embedded Map OSM.
7. **Admin Sales CRM:** Menu tersembunyi (`/admin-inbox`) untuk melacak prospek masuk.
